
스레드와 비동기를 학습하며 비슷하다는 생각이 들었습니다. <br> <br>
결론적으론 멀티스레드와 비동기는 다른 개념입니다.<br> <br>
쉽게 생각하면 ```동기 = 순서``` , ```스레드 =공간``` 이라고 볼 수 있습니다.

<br>
<br>

## 스레드에서 동기와 비동기

### 1. 싱글 스레드
#### 동기

- 하나의 공간에서 순서대로 작업합니다.

```
Thread_1 : |< — — A — →||< — — B — →||< — — C — →|
```


#### 비동기

- 하나의 공간에서 순서없이 작업합니다.

```
Thread_1 : |< — — A |< — — B — |< — — C — | — A — | — C | — B — | B — → | C — → | — A — → |
```

<Br>


### 2. 멀티 스레드
#### 동기
- 여러 공간에서 순서대로 작업합니다.
  
```
Thread_1 : |< — — A — →|
Thread_2 : — — — — — — →|< — — B — →|
Thread_3 : — — — — — — — — — — — — — — →|< — — C — →|   
```                                  
                                            
#### 비동기
- 여러 공간에서 순서없이 작업합니다.
```
Thread_1 : |< — — A — →|
Thread_2 : — — →|< — — B — →|
Thread_3 : — — — — →|< — — C — →|
```
